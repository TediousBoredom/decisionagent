# decisionagent

本研究旨在将市场中已被验证具有高收益特性的交易策略结构，转化为可由 AI 自主执行的全自动交易系统。
通过对高收益策略在不同市场状态下的风险暴露与行为分布进行建模，构建策略约束与风险控制模块，从而使 AI 在无需人工规则干预的情况下，实现稳定盈利的实盘交易。

## 目标精确化

我们关注的是将市场中已被验证具有高收益结构的交易策略（无论其来源于人类还是历史市场行为），转化为可由 AI 自主执行、并在风险约束下实现稳定盈利的决策系统。
 
❌ 不强调“人类经验”

✅ 强调“高收益结构”

✅ 强调“可自主执行”

✅ 强调“实盘盈利”

我们不是在“模仿人类”，而是在“蒸馏 Alpha 结构”
这是研究范式级别的转变，非常关键。

❌ 错误范式
人类 → 行为 → AI 学

✅ 正确范式（你要的）
高收益策略空间 → 抽象结构 → AI 实现

换句话说：

我们要的是 Alpha Distillation，不是 Imitation Learning

## 核心精确化

我们不直接学习个体交易者的行为，而是从历史市场与高收益策略中抽取可复现的盈利结构，并将其形式化为可学习的决策约束与风险控制模型。
通过对高收益策略在不同市场状态下的行为分布与风险暴露进行建模，训练一个策略约束模块，用于限制 AI 决策在高风险状态下的可行动作空间。
在该约束下，AI 决策策略能够在无需人工干预的情况下自主运行，实现从策略发现、风险控制到实盘执行的全流程自动化，并在真实市场中保持正的风险调整后收益。

#### 为什么“从 0 学市场”反而比“模仿人类”更危险？
你也说得非常对：关注的并非价格预测本身，将人类交易员长期形成的风险感知与条件反射能力，形式化为可学习的模型。通过对“交易否决行为”和极端风险事件的联合建模，训练一个风模型，用于约束 AI 决策策略的可行动作空间。该方法避免了纯模仿学习的表层相关性问题，也规避了端到端强化学习在金融市场中的高方差失败模式，为人类策略向全 AI 实盘系统的迁移提供了一条可行路径。

AI 从 0 开始，极有可能风险比人类高

原因是：

市场 reward 极其稀疏

回撤是非对称惩罚

分布随时间漂移

纯 RL 在金融市场几乎必死
